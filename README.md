# 🧠 AI Data Cleaning App

A smart web application that automatically cleans and preprocesses datasets using Python and Streamlit.

---

## 📌 Overview  
This project helps users quickly clean messy datasets by automating common data preprocessing tasks like removing duplicates, handling missing values, and standardizing formats.

---

## ✨ Features  
✔️ Upload Excel files  
✔️ Remove duplicate rows  
✔️ Handle missing values  
✔️ Standardize column names  
✔️ Format date columns  
✔️ Clean numeric data  
✔️ Generate cleaning report  
✔️ Display cleaned dataset  

---

## 🛠️ Tech Stack  
- 🐍 Python  
- 📊 Pandas  
- 🌐 Streamlit
